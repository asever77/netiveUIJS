Examples

Step Count

step 형식은 현재 소수점 숫자는 아직 지원 안합니다.(정수만 가능)
','콤마를 자동 생성합니다.

504025

Slide Count

slide 형식은 소수점도 지원합니다.

5040.25

HTML Structure

basic type


<div class="ui-count" id="exeCount1">504025</div>

Javascript

$plugins.uiCountStep


$plugins.uiCountStep({ 
	id:'exeCount1', 
	value: 504025 
});

$plugins.uiCountSlide


$plugins.uiCountSlide({ 
	id:'exeCount2', 
	value: 5040.25 
});

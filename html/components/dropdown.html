Dropdown

Dropdown은 기본적으로 토글방식으로 문맥상에서 추가적인 콘텐츠나 메뉴에 사용
기본적으로 12가지 위치를 제공하고 있으며 위치에 따른 자동설정도 가능하다.
크게 두가지로 하나는 문장의 흐름에 맞추어 위치한 경우와 모달팝업과 같이 위치가 밖에 있는 경우가 있다.
datePicker 사용 시 필수로 필요한 compontent 요소이다.

dropdown 기본 옵션

옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-drop의 아이디명 기입
ps: [string] 'bl' 총 12가지 위치 설정
  • tl : 상단 좌측정렬
  • tc : 상단 중앙정렬
  • tr : 상단 우측정렬
  • bl : 하단 좌측정렬
  • bc : 하단 중앙정렬
  • br : 하단 우측정렬
  • rt : 우측 상단정렬
  • rm : 우측 중앙정렬
  • rb : 우측 하단정렬
  • lt : 좌측 상단정렬
  • lm : 좌측 중앙정렬
  • lb : 좌측 하단정렬
eff: [string] 'base' 총 6가지 나타나는 효과 설정
  • base : show & hide 효과
  • fade : opacity 효과
  • st : 위에서 아래로 슬라이딩
  • sb : 아래에서 위로 슬라이딩
  • sl : 좌측에서 우측로 슬라이딩
  • sr : 우측에서 좌측로 슬라이딩
offset: [boolean] false 드롭다운 콘텐츠의 위치에 따라 설정
  • true : 위치값 절대값(offset)
  • false : 위치값 상대값(position)
dim: [boolean] false 오브젝트 뒤에 dim처리 설정
auto: [boolean] false 현재 y축 위치에 따라 위아래 위치 자동 설정
openback: [function] false 오픈 시 콜백
closeback: [function] false 닫을 시 콜백
close: [boolean] true 다른 드롭다운 닫힐지 여부
  • true : 선택된 이외의 드롭다운은 닫힘
  • false : 변경없음
focushold: [boolean] true 포커스 홀드 여부
  • true : 'hold' 포커스 홀드
  • false : 'sense' 포커스 체크

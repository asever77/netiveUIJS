옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) 코딩리스트를 적용 시킬 영역(ui-codinglist)의 id 명
url: [string] 'link' (필수) 코딩리스트 정보를 불러올 json or html 파일 경로
type: [string] 'json' 문서타입 설정. 기본으로 json 문서 타입 설정됨
  • 'json'
  • 'html'

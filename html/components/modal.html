Javascript

$plugins.uiModal


// width,height 값을 주면 
$plugins.uiModal({ 
	id:'modal_name', 
	width:500, 
	height:500, 
	ps:'top' 
	endfocus:'idname', 
	full:true, 
	link:'/netiveUI/html/modal/sample.html', 
	openback: function() { console.log('open callback'); },
	closeback: function() { console.log('close callback'); } 
});
$plugins.uiModal 옵션 설명

옵션명 유형 기본 설명
id: [string] 'modal_name' (필수) ui-modal의 아이디명 기입
ps: [string] 'center' 모달 레이어 팝업 생성될 위치 설정
  • 'top' : 상단 위치
  • 'center' : 중앙 위치
  • 'bottom' : 하단 위치
open: [boolean] true 실행 시 바로 모달 레이어 팝업 오픈여부 선택
  • true : 모달 레이어 팝업 오픈
  • false : 모달 레이어 팝업 관련 파일만 불러온 상태
    $plugins.uiModalopen({ id:'modal_name' }); 을 실행해야지 모달 열림
full: [boolean] false 전체 사이즈 모달 레이어 팝업 설정
  • true : 전체화면
  • false : 기본 사이즈
link: [string] Ajax로 파일을 불러올 url경로
endfocus: [string] document.activeElement 모달 레이어 팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달 레이어 팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정
width: [number] 모달 레이어 팝업 가로 크기
옵션이 없을 경우 모달 팝업의 크기
height: [number] 모달 레이어 팝업 세로 크기
옵션이 없을 경우 모달 팝업의 크기
openback: [function] false 모달 레이어 팝업 오픈 시 콜백값 실행
closeback: [function] false 모달 레이어 팝업 닫을 시 콜백값 실행
autofocus: [boolean] true 모달 닫을 시 포커스 자동이동 여부 설정
mpage: [boolean] false 모바일 디바이스에서 true일 경우 페이지 형식(전체 화면)으로 생성
cutline: [number] 31 .ui-modal-cont 의 기본 페딩이 15일 경우 그 값보다 1 큰값으로 설정하여 콘텐츠 로딩여부 판단에 사용.

$plugins.modal.system


// 확인 
$plugins.modal.system({ 
	type : 'alert',
	btn_alert : '확인',
	cont_text : '확인하세요.',
	state : '알림',
	confirmCallback: function(){console.log('alert 확인')} 
});

// 선택 
$plugins.modal.system({ 
	type : 'alert',
	btn_alert : '확인',
	cont_text : '오류가 있습니다.',
	state : '오류',
	confirmCallback: function(){console.log('alert 확인')} 
});

// 오류 
$plugins.modal.system({ 
	type : 'confirm',
	btn_confirm_yes : '확인',
	btn_confirm_no : '취소',
	cont_text : '둘중에 하나를 선택하세요. <br> 오직 하나만~',
	state : '선택',
	confirmCallback: function(){console.log('confirm 확인')},
	cancelCallback: function(){console.log('confirm 취소')} 
});

$plugins.uiModalOpen

$plugins.uiModal (open:false) 옵션을 준 상태 모달 레이어 팝업 오픈 


$plugins.uiModalOpen({ id:'modal_name' });

$plugins.uiModalResize

모달 레이어 팝업안에 컨텐츠의 동적 생성으로 사이즈 변경이 필요할 시 실행


$plugins.uiModalResize({ id:'modal_name' });

$plugins.uiModalClose

모달 레이어 팝업 닫을 시 사용


$plugins.uiModalClose({ 
	id:'modal_name', 
	endfocus:'idname', 
	callback:function(v){console.log(v)} 
});
$plugins.uiModalClose 옵션 설명

옵션명 유형 기본 설명
id: [string] 'modal_name' (필수) ui-modal의 아이디명 기입
remove: [boolean] false 모달팝업 콘텐츠 삭제 여부
  • true: 코드삭제
  • false: 미삭제
endfocus: [string] document.activeElement 모달팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정
callback: [function] 모달팝업 닫을 시 콜백함수 실행

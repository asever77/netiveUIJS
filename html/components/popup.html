popup

window popup을 생성할때 사용

Examples

팝업띄우기 팝업띄우기

HTML Structure


<a href="..." class="btn-base" id="uiPopupA" target="_blank" aria-haspopup="true" title="새 창 열림">팝업</a>

Javascript

$plugins.uiPopup


$('#uiPopupA').on('click', function(e){
	e.preventDefault()
	$plugins.uiPopup({
		link: this.href,
		name: 'new popup',
		width: 790,
		height: 620,
		align: 'center',
		top: 0,
		left: 0,
		toolbar: 'no',
		location: 'no',
		memubar: 'no',
		status: 'no',
		resizable: 'no',
		scrolbars: 'yes'
	});
});

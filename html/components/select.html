기본 스타일

Custom Select

모든 브라우저에서 동일한 UI와 디자인으로 접근 가능. option도 디자인 표현이 가능

지역선택

HTML Structure


<label for="uiSel1" class="ui-select-label">아이유 앨범</label>
<div class="ui-select">
	<select title="Palette 선택" id="uiSel1">
		<option value="1">밤편지</option>
		<option value="2">잼잼</option>
	</select>
</div>

Javascript

$plugins.uiSelect()

현재 select box기준으로 디자인 select 생성. 첫적용이나 동적인 변경이 있을때 사용한다. 특히 동적인 변경일 경우는 id로 개별 실행해주는게 좋다.


//전체 실행
$plugins.uiSelect();

//개별 실행
$plugins.uiSelect({ 
	id:'forSelLocal', 
	current:0 
});
selectg 기본 옵션

옵션명 유형 기본 설명
id: [string] 'name' ui-select에 포함된 select의 아이디명 기입
current: [number] null option 선택될 위치
callback: [function] false 값 선택 시 콜백실행.

$plugins.uiSelectAct()

마우스, 키보드 이벤트가 아닌 option값 선택을 할 때 사용. $plugins.uiSelect()을 사용해도 되나 불필요한 재구성을 하지 않아 값변경 시에는 $plugins.uiSelectAct를 사용한다. 


$plugins.uiSelect({ 
	id:'forSelLocal', 
	current:0,
	callback:function(v){console.log(v)}
});

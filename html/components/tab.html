HTML Structure


<div class="ui-tab" id="exeTab1">
	<div class="ui-tab-btns">
		<button type="button" class="ui-tab-btn"><i>탭1</i></button>
		<button type="button" class="ui-tab-btn"><i>탭2</i></button>
	</div>
	<div class="ui-tab-pnls">
		<section class="ui-tab-pnl">
			<h1 class="ui-tab-tit">탭1</h1>
			탭1 내용
		</section>
		<section class="ui-tab-pnl">
			<h1 class="ui-tab-tit">탭2</h1>
			탭2 내용
		</section>
	</div>
</div>

Javascript

$plugins.uiTab


$plugins.uiTab({ 
	id:'exeTab1', 
	current:0, 
	label:'인증서', 
	callback:function(v){...} 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-tab의 아이디명 기입
current: [number] 0 처음 열린패널 선택
label: [string] '' aria-label 명
unres : [boolean] false 탭 선택 시 컨텐츠 변경 여부
  • true : 미변경
  • false : 변경
callback: [function] function 콜백값으로 아이디와 현재선택 index값

$plugins.uiTabAct


$plugins.uiTabAct({ 
	id:'exeTab1', 
	current:0, 
	callback:function(v){...} 
});

id: [string] 'name' (필수) ui-tab의 아이디명 기입
current: [number] 0 처음 열린패널 선택
unres : [boolean] false 탭 선택 시 컨텐츠 변경 여부
  • true : 미변경
  • false : 변경
callback: [function] function 콜백값으로 아이디와 현재선택 index값 

$plugins.uiTab({ 
	id:'exeTab1', 
	current:0, 
	unres:true,

});

parameter 값으로 탭 선택방법


single 선택
/netiveUIJS/html/components/tab.html?tab=1 

복수 선택
/netiveUIJS/html/components/tab.html?tab=exeTab1*0+exeTab1_2*1+exeTab2*1 
탭아이디*선택+탭아이디*선택....
주의 : tab의 아이디를 '*'이 들어가는 이름으로 설정하면 안된다.

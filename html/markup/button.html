Button Base

기본적으로 <btn-wrap> </btn-wrap>로 버튼요소를 감싸줍니다. 버튼태그로는 <a>, <button> 태그를 사용 하며,
<a>는 링크요소버튼으로 'target="" ', 'title="" '속성을 함께 사용하고, <button>는 페이지내에서 이뤄지는 동작관련 버튼으로써 사용한다.

a 버튼
a 버튼 

<div class="btn-wrap">
	<button class="btn-base-m" type="button"><span>버튼</span></button>
	<a class="btn-base-m" href="#none"><span>버튼</span></a>

	<button class="btn-base-m disabled" type="button" disabled><span>버튼</span></button>
	<a class="btn-base-m disabled" href="#none"><span>버튼</span></a>
</div>

Button Size

버튼의 사이즈는 크게 3가지로 나눈다.btn-base-s, btn-base-m, btn-base-l를 추가하면 됩니다.

small

.btn-base-s(a)

midium(base)

.btn-base-m(a) .btn-base(a)

large

.btn-base-l(a) 

<div class="btn-wrap">
	<button class="btn-base-s" type="button"><span>버튼s사이즈</span></button>
	<a class="btn-base-s" href="#none"><span>버튼s사이즈</span></a>
	
	<button class="btn-base-m" type="button"><span>버튼m사이즈</span></button>
	<a class="btn-base-m" href="#none"><span>버튼m사이즈</span></a>

	<button class="btn-base-l" type="button"><span>버튼l사이즈</span></button>
	<a class="btn-base-l" href="#none"><span>버튼l사이즈</span></a>
</div>

Button 좌우정렬

버튼의 좌우정렬 기본은 중앙정렬이며, 부모요소에 좌측정렬은 txt-l,클래스를, 우측정렬은 txt-r 클래스를 추가하면 됩니다.

a 버튼 좌측
a 버튼 센터
a 버튼 우측 

<div class="btn-wrap txt-l">
	<button class="btn-base" type="button"><span>버튼 좌측</span></button>
	<a class="btn-base" href="#none"><span>버튼 좌측</span></a>
</div>

<div class="btn-wrap">
	<button class="btn-base" type="button"><span>버튼</span></button>
	<a class="btn-base" href="#none"><span>버튼</span></a>
</div>

<div class="btn-wrap txt-r">
	<button class="btn-base" type="button"><span>버튼 우측</span></button>
	<a class="btn-base" href="#none"><span>버튼 우측</span></a>
</div>

Button Style

버튼의 기본 스타일 외의 스타일을 줄 경우 btn-base 를 제외한 btn-name(btn-close, btn-del, btn-next ...) 으로 쓴다.


<button class="btn-close" type="button"><span>닫기</span></button>

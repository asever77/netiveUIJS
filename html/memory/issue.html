  • if, for 구문등 {...}안에 함수작성 시 오류 발생

    자바스크립트에서 'use strict' 모드일때 if, for 구문안에 함수를 작성하면 ie에서 오류 체크 됨. 함수는 구문 안에 작성안함.

    #자바스크립트 #구문 오류
  • ie8에서 aria로 선택한 스크립트 변경사항을 바로 체크 못함

    class로 같이 설정하여 변경해줘서 해결

    #ie8 #aria #자바스크립트
  • 아이프레임 안에 있는 전화번호 터치 시 IOS에서는 동작안하는 증상

    아이프레임으로 인한 문제 발생으로 터치 시 부모창의 로케이션 링크값으로 해결
    예) window.parent.loacation.href='tel:021234567';

    #ios #tel #iframe
  • z-index 값을 명확히 기입하자.

    모바일환경에서 동작 시 값이 없는 경우 잔상이 생기는 경우가 있다.

    #모바일 #z-index #잔상
  • 파이어폭스 버튼 여백 생기는 문제발생 시

    button::-moz-focus-inner{padding:0l birder:0}

    #파이어폭스 #버튼 #여백
  • placeholder 텍스트를 스크린리더에서 읽어주지 못함
    #접근성 #placeholder #스크린리더
  • [class^="name"].classname

    html에서 정의된 class의 순서와 다르면 적용이 안되는 문제가 있다.
    [class^="name"] 대신 [class*="name"] 로 하면 해결이 된다.

    #class #^= #순서
  • IE에서 개발자모드 오류날때

    IE11-Windows6.1-KB2987107-x64 를 설치하면 된다.

    #ie #개발자모드

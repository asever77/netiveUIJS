Base z-index

기본적인 z-index 값을 정하여 체계적인 위치설정

옵션 설명
class명 옵션명 설정값 설명
.base-skip z-index 100 스킵네비게이션 메뉴 (모든 컨텐츠의 최상위에 존재)
.base-layer z-index 20 모달, 드롭다운, 툴팁 등 레이어로 뜨는 오브젝트(본문보다 상위)
.base-wrap z-index 0 본문을 감싸는 전체 영역(header,main,footer 등)

Base Layer z-index

기본적인 z-index 값을 정하여 체계적인 위치설정

옵션 설명
class명 옵션명 설정값 설명
.base-layer .modal-backdrop z-index 0 모달의 암막레이어
.base-layer .ui-modal z-index 1~99 모달
.base-layer .ui-dropdown z-index 100~199 드롭다운
.base-layer .ui-tooltip z-index 200~299 툴팁

class 작명

class 작명 일반 규칙

class명은 cabab casing(케밥 표기법)으로 영문 소문자, 숫자, 하이픈(-)으로 작성한다.

시작문자는 영문 소문자로만 시작한다.


.btn-base
.list2

class 작명 조합 규칙

  • 하이픈(-)은 2개 이상의 단어 조합할 때만 사용한다.
  • 하이픈(-)조합은 최대 2단계를 초과하지 조합한다.
  • 단어와 숫자 조합은 하이픈(-)을 생략한다.
  • 숫자는 1부터 시작하며 '01' 과 같이 앞에 '0'을 붙여 사용하지 않는다.

.btn-base-large
.box-dark
.n1

class 조합 규칙

  • class의 조합은 3개 이하 및 우선순위 점수 33점 미만을 권장한다.
  • class, 가상선택자(10점), 엘리먼트(1점)
  • id의 사용은 최대한 제외한다.

.btn-base .on

class 작명 자바스크립트 전용

ui- 로 시작하는 클래스는 자바스크립트에서만 사용한다.


.ui-name

class 예약어

기본적으로 주로 사용하는 class에 대해 구분지어 예약어를 사용한다.

class 예약어
예약 접두어 설명
.base- 전체적으로 기본적으로 사용하는 스타일

  • .base-layer : 공통 모달팝업, 드룹다운, 툴팁 등 base-wrap 상위로 뜨는 콘텐츠 공간

  • .base-body : header, footer를 제외한 영역
  • .base-main : main 영역(모든 문서에 반복적으로 있는 어떠한 컨텐츠라도 제외되어야 하며 문서 안에 가장 유일한 요소여야 합니다.)
  • .base-aside : aside 영역
  • .base-content : 주요 내용이 들어가는 영역
  • .base-footer : footer 영역

.box- box에 관련된 스타일

.pop- popup 관련 스타일
.ui- 자바스크립트 관련 클래스

자바스크립트 작명

자바스크립트 변수명

변수명은 snake casing(스네이크 표기법)으로 영문 소문자, 숫자, 하이픈(-)으로 작성한다.

시작문자는 영문 소문자로만 시작한다.


var num = 0;
var num_box = true;

자바스크립트 함수명

변수명은 camel casing(카멜 표기법)으로 작성한다.

시작문자는 영문 소문자로만 시작한다.


function creatNext() {....}
var creatPrev = function() {.......}

ID 작명

ID 작명 규칙

이름은 camel casing(카멜 표기법)으로 작성한다.

이름의 시작문자는 영문 소문자로만 시작한다.

ID에는 어떠한 스타일도 주지 않는다.


#btn
#btnClose1
